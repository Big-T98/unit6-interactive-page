# Live Profile Card - Unit 6 Project

My Unit 6 JavaScript project. I built an interactive web page that responds to user input in real time using:

- **`getElementById`** to grab elements
- **`addEventListener`** for events (input, change, click)
- **`textContent`** to update the card display
- **`.value`** to read what the user typed
- **`classList.toggle`** for dark mode
